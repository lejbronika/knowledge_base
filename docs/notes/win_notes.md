# Windows Notes 

## Delete Windows.old

Чтобы удалить предыдущую версию Windows с экрана загурзки необходимо выполнить следующие действия:

- запустить консоль от имени администратора
- `bcdedit`
- `bcdedit /delete {win_id} /f`
